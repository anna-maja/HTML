JavaScript - om variabler

Code-convention

Variabler får börja med bokstav, understreck, dollartecken.
I JavaScript är variablerna inte typiska. Dvs inte "int", "double", "String" etc.
Vi använder bara namnet "var" istället, och innehållet tolkas av systemet.
Använd inte reserverade ord som variabelnamn.

Innehåll med "text text" i dubbelfnuttar, tolkas som sträng.
Innehåll utan dubbelfnutt tolkas som variabelnamn!!!
