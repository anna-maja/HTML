Containerns bredd hanteras med grid-systemet. Automatiskt så att marginaler t ex tas bort helt när skärmen är liten.

Grid system - kolumner

Det finns alltid 12 kolumner i bootstrap grid system.
Standard brytpunkter för olika typer av skärmar.

Följande förkortningar används:
xs för mobiltelefoner < 576px
sm för mindre surfplattor <= 576px
md för medelstora skärmar <= 768px
lg för stora skärmar <= 992px
xl för extra stora skärmar <= 1200px
