Hex-kod

Baserat på principen att värden alltid anges rgb == red, green, blue.
Alltid samma ordning, och därefter gradering inom respektive färg.

F = maxvärde.
00F = 0 red, 0 green, max blue.
FFF = maxvärde på alla färger == vit.
000 = min på alla färger == svart.
