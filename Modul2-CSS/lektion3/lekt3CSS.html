Lektion 3, modul 2: CSS

Allmänt om CSS

CSS == ”Cascading Style Sheets”
Innehåll, struktur --- HTML
Layout, design, visning, interaktivitet --- CSS
1) hjälper till i sökoptimering
2) innehållet mer renodlat == mer relevant i text-till-tal.

Än så länge vet vi alltså om två nivåer:
i grunden - HTML, grunden, strukturen.
nästa lager - CSS, design, yt-funkionalitet.
(sen kommer fler lager, men dit kommer vi senare.)
== Separation of concerns. Uppdelning i olika lager / innehåll,
och därmed också möjlighet till uppdelning på olika personer / specilaieterer.
